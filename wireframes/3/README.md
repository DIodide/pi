# Princeton Intelligence Slash3 Mockups

This set keeps the Slash2 visual direction but narrows the product scope to the current TigerApps PI plan.

## Pages

- `index.html` - v3 hub and product scope summary.
- `home.html` - daily command center for NetID-signed students.
- `chat.html` - core connector chat with app-native renders and action confirmation.
- `connectors.html` - TigerApps connector catalog and MCP readiness.
- `course-workflow.html` - academic planning across PrincetonCourses, TigerJunction, TigerPath, and TigerSnatch.
- `campus-workflow.html` - The Forum and future listserv email pipeline.
- `draw-workflow.html` - TigerDraw planning, marked as MCP soon.
- `permissions.html` - NetID scopes, confirmation rules, and audit log.

## Product Scope

Included: PrincetonCourses, TigerJunction, TigerPath, TigerSnatch, The Forum/listservs, TigerDraw.

Excluded: TigerVideos, PrincetonQuest, TigerSpot, TigerType.
